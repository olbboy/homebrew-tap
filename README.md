# Homebrew Tap

Custom Homebrew formulae by olbboy.

## Installation

```bash
brew tap olbboy/tap
```

## Formulae

| Formula | Description |
|---------|-------------|
| [claudekill](Formula/claudekill.rb) | Find and delete .claude folders to reclaim disk space |

### claudekill

```bash
brew install olbboy/tap/claudekill
```

Or:

```bash
brew tap olbboy/tap
brew install claudekill
```
